# 🚖 Uber Data Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Uber ride booking data and generate meaningful business insights. The dashboard helps visualize booking trends, revenue performance, customer behavior, vehicle usage, and payment analytics.

---

## 📊 Dashboard Features

* Total Revenue Analysis
* Completed vs Lost Bookings
* Vehicle Type Performance
* Monthly & Quarterly Booking Trends
* Customer & Driver Ratings
* Payment Method Analysis
* Ride Cancellation Reasons
* Customer Revenue Insights

---

## 🛠 Tools & Technologies Used

* Power BI Desktop
* Excel / CSV Dataset
* Data Cleaning & Transformation
* DAX Calculations
* Data Visualization

---

## 📈 Key Insights

* 💰 Total Revenue Generated: **52M+**
* ✅ Completed Bookings: **149K**
* ❌ Lost Bookings: **57K**
* 📍 Total Distance Covered: **2.51M**
* 🚕 Auto rides generated the highest revenue
* 💳 UPI was the most used payment method

## 🎯 Objective

The objective of this project is to analyze ride-booking data and provide actionable insights that can help improve customer experience, operational efficiency, and revenue generation for ride-sharing businesses like Uber.

This dashboard demonstrates how Power BI can transform raw ride-booking data into interactive business insights through effective data visualization and analytics.
